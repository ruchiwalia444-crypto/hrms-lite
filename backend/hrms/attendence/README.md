# HRMS Lite

A lightweight Human Resource Management System to manage employees and track attendance.

## Tech Stack

Backend:
- Python
- Django REST Framework

Frontend:
- React

Database:
- SQLite

## Features

- Add employees
- View employees
- Delete employees
- Mark attendance
- View attendance records

## Run Locally

Backend:
cd backend
python manage.py runserver

Frontend:
cd frontend
npm start
